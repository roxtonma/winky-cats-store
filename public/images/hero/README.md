# Hero Images

This directory contains hero section promotional images for the home page.

## Current Files
- `hero-placeholder.svg` - Desktop hero background (1920x600px)
- `hero-mobile-placeholder.svg` - Mobile hero background (768x600px)

## Recommended Image Specifications

### Desktop Hero Image
- **Dimensions**: 1920x600px (or 2x: 3840x1200px for retina)
- **Aspect Ratio**: 16:5 (widescreen banner)
- **Format**: WebP (with JPG fallback)
- **File Size**: Under 200KB (optimized)
- **Content**: Product lifestyle shots, collection highlights, or promotional content

### Mobile Hero Image
- **Dimensions**: 768x600px (or 2x: 1536x1200px for retina)
- **Aspect Ratio**: 6:5 (taller for mobile viewport)
- **Format**: WebP (with JPG fallback)
- **File Size**: Under 150KB (optimized)
- **Content**: Cropped/adapted version of desktop image

## Content Suggestions
- **Product lifestyle photography**: T-shirts being worn in casual settings
- **Collection highlights**: Feature new arrivals or bestsellers
- **Promotional content**: Seasonal sales, special offers
- **Brand personality**: Incorporate Winky Cats theme with playful elements

## Image Guidelines
1. Ensure text overlay area (center-left) has adequate contrast
2. Keep important content away from edges for safe zones
3. Test readability of white text overlay
4. Use consistent brand colors (#FF6B9D, #FF1F7F)
5. Optimize images before deployment

## Replacing Placeholders
Simply replace the placeholder SVG files with optimized WebP/JPG images using the same filenames:
- `hero-placeholder.webp` (or .jpg)
- `hero-mobile-placeholder.webp` (or .jpg)

The code will automatically use the new images.
